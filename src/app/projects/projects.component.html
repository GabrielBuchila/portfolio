SList.

Platform: iOS.
Tools & Technologies: Xcode, Swift Programming Language.
Objective:This project was developed as a learning exercise to explore the Xcode IDE and the Swift programming language, driven by curiosity about iOS app development. The goal was to create a simple yet functional iOS application that allows users to manage multiple shopping lists, where each list can contain a variety of items.
Project Overview:The Shopping List Manager app is a user-friendly iOS application that enables users to create and manage multiple shopping lists, each containing multiple items. The app provides a straightforward interface for adding, editing(in progress), and deleting lists and items, making it easy for users to organize their shopping needs.
